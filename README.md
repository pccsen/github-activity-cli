## Project Description

This is a simple command-line tool (CLI) that fetches the recent activity of any GitHub user and displays it in the terminal.

The tool allows you to:

- Provide a GitHub username as a command-line argument.
- Fetch events such as pushed commits, opened/closed issues, pull requests, and starred repositories.
- Handle errors gracefully (invalid username, connection issues, etc.).
